StackSet class

class smokestack.StackSet(out: IO[str])

A set of stacks.

abstract property stacks: List[Type[smokestack.stack.Stack]]

Gets the stacks in this set.

In this example, the “Application” set includes an application, database and logging stack:

from smokestack import StackSet

import myproject.stacks


class ApplicationStackSet(StackSet):

    # ...

    @property
    def stacks(self) -> List[Type[Stack]]:
        return [
            myproject.stacks.ApplicationStack,
            myproject.stacks.DatabaseStack,
            myproject.stacks.LoggingStack,
        ]

Note that the deployment order is prescribed by each stack’s smokestack.Stack.needs property.
